JSXTransformer.js lets us write html code in javascript, it is then compiled to javascript in the browser. It seems that JSXTransformer is deprecated, the reason being that the process should rather be done before running the site, for example with a gulp task.

React.createClass

  • It must have a render function
  • Multiple components must have a container element
  • Properties are ways to pass in information when rendering
  • React wants properties as camelCase in render method, not snake case like Angular
  • It is optional to pass them from the render method
  • Can have children inside rendered tag, there is a property name this.props.children to get to them
  • Can use evets
  • onClick and also className should not have quoutes if binding to property
