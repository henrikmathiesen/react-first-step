React With Webpack workflow

Setting up a React development environment with Webpack (the new kid on the block)

  • No Bower
  • No Gulp
  • No .babelrc
  • React downloaded via NPM, saved as runtime dependency (NPM is prefered over Bower with this workflow)
  • We use npm to run webpack see package.json under scripts
  • We need a webpack config file, paths and presets are defined there
  • We compile out code with npm run webpack in console

package.json for this workflow

"devDependencies": { "babel-core": "^6.10.4", "babel-loader": "^6.2.4", "babel-preset-es2015": "^6.9.0", "babel-preset-react": "^6.11.1", "webpack": "^1.13.1" } "dependencies": { "react": "^15.2.0", "react-dom": "^15.2.0" } }
